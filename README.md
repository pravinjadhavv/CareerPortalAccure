🚀 Career Portal Application

A full-stack Career Portal web application built using ReactJS, Spring Boot, and MySQL. This platform allows candidates to apply for jobs, companies to post job openings, and admins to monitor platform activity.

📌 Features

👤 Candidate

Register & Login (JWT Authentication)

Browse available jobs

Apply for jobs

Upload resume

Edit profile

🏢 Company

Register & Login

Post job openings

View applicants

Manage company profile

🛠️ Admin

View candidate reports

View company reports

Monitor system data

🧑‍💻 Tech Stack

Frontend:

ReactJS (Vite)

TypeScript

Tailwind CSS

Backend:

Java

Spring Boot

Spring Security (JWT)

Database:

MySQL

⚙️ Setup Instructions

🔹 Backend Setup

cd backend/career-portal
mvn clean install
mvn spring-boot:run

🔹 Frontend Setup

cd frontend
npm install
npm run dev

🔐 Authentication

JWT-based authentication
